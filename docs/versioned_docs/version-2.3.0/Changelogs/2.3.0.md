# 2.3.0

## Changes
### New Conditions
- [Y Level](../Rules/Conditions/YLevel.md): Match deaths based on the player's Y coordinate (height) at the time of death.
- [Light Level](../Rules/Conditions/LightLevel.md): Match deaths based on the light level at the player's location at the time of death.
- [Death Cause](../Rules/Conditions/Cause.md): Match deaths based on what killed the player (fall, lava, mobs, etc.).

### New Effects
- [Command Execution](../Rules/Effects/Command.md): Execute commands as the player dies or respawns.
- [Lightning Strike](../Rules/Effects/Lightning.md): Summon a lightning bolt at the player's location on death.

## Bugfixes
- Empty localization string now properly suppresses message
- Fixes damage effect applying extra components to items without durability (breaks stacking)
