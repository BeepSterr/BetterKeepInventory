## About

Do you want to use the KeepInventory gamerule but dislike how it completely removes any risk with death? With this plugin you get to choose how punishing death should be!
Using a rich rule-based configuration system you get fine control over exactly what happens to players when they die!

## Features
- ✅ **Rich rule-based configuration system**
- ✅ **Use Permissions, Worlds or [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi) conditions**
- ✅ **Make items drop**
- ✅ **Control XP Level**
- ✅ **Apply Durability Damage**
- ✅ **Re-apply hunger level**
- ✅ **Take Money from players**
- ✅ **Expandable via the [Developer API](https://github.com/BeepSterr/BetterKeepInventory?tab=readme-ov-file#developer-api)**
- ✅ **Supports Spigot, Paper & Folia**
