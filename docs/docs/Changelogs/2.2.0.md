# 2.2.0 (Work In Progress)

:::info
This version is currently a work in progress and may be subject to changes before its official release.
You can install pre-release versions from the [Alpha Versions](https://modrinth.com/plugin/betterkeepinventory/versions?c=alpha) page on Modrinth and choosing the latest one.

**Keep in mind that these versions are mostly untested and have a high chance of containing bugs.**
:::

**Changes:**
- Changes to [Resource Lists](../ResourceLists) behavior:
  - MaterialType has been refactored into a new type called ResourceList 
  - Now support `ALL` and `*` to add all items to the list
  - Now support prefixing with `!` to remove an item from the list
    - This also works for Resource Groups (`G:`) 
  - Now support using `mineraft:oak_log` type ids

- Documentation has been moved from Notion to a static site generator using Docusaurus
  - You can find the new documentation [here](https://bki.beepsterr.com)
